Spalné teplo

Zpět na předmět C1040

Zadání

Vypočítejte spalné teplo C2H2, jsou-li dána slučovací tepla reakcí:

a) 2 C (s) + H2 (g)   →C2H2 (g)  ΔH = 226,92 kJ.mol-1
b) C (s) + O2 (g)   →CO2 (g)  ΔH = −393,97 kJ.mol-1
c) H2 (g) + ½ O2(g)   →H2O (l)  ΔH = −285,96 kJ.mol-1

Řešení

Pro výpočet budeme potřebovat rovnici hoření acetylenu.

C2H2 (g) + 2,5 O2 (g) → 2 CO2 (g) + H2O (l)

Tuto rovnici potřebujeme získat lineární kombinací tří rovnic ze zadání. Pro náš příklad odvodíme tento výraz:

-a + 2b +c

Získáme tak rovnici:

C2H2 + 2 C + 2 O2 + H2 + ½ O2 → 2 CO2 + H2O + 2 C + H2

C2H2 + 2 O2 + ½ O2 → 2 CO2 + H2O

Entalpii reakce pak vypočítáme dosazením slučovacích entalpií:

-a + 2b +c = −226,96 + 2.(−393,97) + (−285,96) = 1300,86 kJ.mol-1

Spalné teplo acetylenu je 1300,86 kJ.mol-1.
